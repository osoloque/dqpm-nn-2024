# This repository contains data from preprint arXiv: 2102.06660v1

1.)
# Results from the polynomial fits in Fig. 7 are listed in kappa_all.dat
# with the format:
# T[MeV] k2B2 error k4B2 error k2BS error k4BS error k2S2 error k4S2 error

2.)
# Data for chiB2 and its T derivative in Fig. 8 are listed in chiB2.dat
# with the format:
# T[MeV] chiB2 error T d/dT(chiB2) error 

3.)
# Data for thermodynamic observables in Figs. 9-10 are listed in thermo_k4.dat
# (in the case with k4!=0) and thermo_nok4.dat (in the case with k4=0)
# with the format:
# muB/T T[MeV] p/T^4 error nB/T^3 error s/T^3 error e/T^4 error nS/T^3 error chiS2 error
# where:
# p = pressure
# nB = baryon density
# s = entropy density
# e = energy density
# nS = strangeness density
# chiS2 = second strangeness susceptibility
